똥 피하기 (Avoid the poop)
==========================

Change character directory path
-------------------------------
+ background
+ character
+ ddong

How to play
-----------
+ Use directional key Left and Right.
+ Move the character and avoid the falling poop.
  + poop falls in random place.
+ If the character and the poop hit, Game Over.
